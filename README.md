## Instalação no Linux

```
git clone https://github.com/joaostack/neovim-config ~/.config/nvim
```

```
nvim
```

O servidor LSP para C# vem pré instalado, caso queira remover, delete os seguintes arquivos:\
[lua/plugins/roslyn_filewatch.lua](https://github.com/joaostack/neovim-config/blob/main/lua/plugins/roslyn_filewatch.lua)\
[lua/plugins/roslyn.lua](https://github.com/joaostack/neovim-config/blob/main/lua/plugins/roslyn.lua)
